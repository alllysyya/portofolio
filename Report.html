My experience creating a website

My experience in creating a website started when I leamed basic HTML and CSS at school. At first, I felt confused because there were mary tags and codes to remember. However, I continued practioing and tried to make a simple website step by step, I learned how to create headings, paragraphs, images, and navigation menus.

During the coding process, I faced several challenges such as fixing layout errors and choosing suitable colors for the website deaign. Sometimes the code did not work properly, and I had to check it many times, Even though it wes difficult, I enjoyed the process because I could improve my problem solving skills and become more patient while learning programming.

I also learned how to make the website look more attractive by adding animations, backgrounds, and better layouts, I tried combining HTML, 2:3, and JaveScript to create a simple but interactive website, From this experience, I understood that teamwork, creativity, and practice are very important in web development.

After finishing the website project. I felt happy and proud of myself. Creating a website gave me new experiences and increased my conftlence in coding. I realized that making a good website requires creativity, practice, and patience. This experience also motivated me to keep learning web development in the future.
